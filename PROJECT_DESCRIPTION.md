# Project Description

## Short Description

A desktop-based computer vision application that performs real-time object detection using pretrained YOLOv5 models through a simple Tkinter interface.

## Extended Description

This project is a Python GUI application that integrates **Tkinter**, **OpenCV**, **PyTorch**, and **NumPy** to create a practical object-detection workflow. The system allows the user to choose from multiple YOLOv5 model variants, process webcam or media-file input, and display detected objects in real time. It also includes an alert mechanism that warns the user when a selected object class appears and a logging feature that saves timestamped results to a text file.

The project is suitable as a mini-project in computer vision, machine learning deployment, or Python GUI development because it demonstrates how deep-learning inference can be connected to a usable desktop front end.

## Core Highlights

- GUI-based control for model selection
- Webcam and custom media input support
- Real-time YOLOv5 object detection
- Object-specific alerting
- Timestamped result export
- Good base for future expansion into a larger vision application
